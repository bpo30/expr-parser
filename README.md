# Expression Parser

A Java library for parsing and evaluating mathematical and bitwise expressions with variables.

## Features

- Supports operations: `+`, `-`, `*`, `/`, `&`, `|`, `^`
- Handles unary minus and parentheses
- Supports integer constants and variables
